# 🌐 Web Scraper Python

Projeto simples que coleta títulos de notícias de um site utilizando Python.

## Tecnologias

- Python
- Requests
- BeautifulSoup

## Como executar

Instale as bibliotecas:

```
pip install requests
pip install beautifulsoup4
```

Execute:

```
python scraper.py
```
